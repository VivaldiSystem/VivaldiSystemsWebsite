Get Started: A Guide to Installing the Demo
Follow POS

Follow POS installs on any new Windows-based computer you throw at it. But to be sure, please check Follow’s system requirements before continuing.

Operating System:
32 or 64 bit on Windows XP, 7, or 10
Memory:
2GB of Random Access Memory (RAM)
Hard Drive:
500MB minimum space

For Advanced Users Only

Step 0: Install Anydesk

Run the downloaded file until setup is completed. Follow this link for more info on how to setup Anydesk.

Step 1: Install Client Software

Run the setup file. Do not open Follow just yet.

Step 2: Install Upgrade

If you have the software running, exit Follow POS before continuing with the upgrade.

Install the upgrade by unzipping the file to the C:\Follow directory

Step 3: Install Sumatra PDF

Install Sumatra PDF to view reports efficiently. It allows to run multiple reports at once without locking out. Bad Adobe!!
