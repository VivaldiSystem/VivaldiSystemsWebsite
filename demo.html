Get Started: A Guide to Installing the Demo
Follow POS

Follow POS installs on any new Windows-based computer you throw at it. But to be sure, please check Follow’s system requirements before continuing.

Operating System:
32 or 64 bit on Windows XP, 7, or 10
Memory:
2GB of Random Access Memory (RAM)
Hard Drive:
500MB minimum space

Follow these steps to continue

Step 1: Install the client software of Follow POS

and the save the primary file

Run the downloaded <setup.msi> file until setup is completed

Step 2: Upgrade Software

We strongly recommended you download the upgrade so we can ensure you have the latest features.

If you have the software running, exit Follow POS before continuing with the upgrade.

Install the upgrade by unzipping the <vivaldi8upgrade.exe> file to the C:\vivaldi8 directory

Step 3: Install Report Viewer

to be redirected to the Microsoft website where you can download the report viewer

Run and install the <ReportViewer.exe>

Once that’s done, open the software and log in as a demo user using email address rick2164@msn.com and password 2164 when prompted.

If you have any issues with the installation, be sure to contact customer support or just send an email at Customer Support.

You’re all done!
