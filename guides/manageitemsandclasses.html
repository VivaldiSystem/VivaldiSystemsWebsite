How to Manage Items and Classes

This tutorial will guide you through the steps on how to manage by adding and modifying your items, classes, and prices.

Follow these steps to continue

When you’re setting up your system, one of the first things you would need to do is setup our classes and items pertaining to those classes before writing a ticket for a customer. Let's first start with classes.

Step 1: Head to Class Manager

Go to "Utilities" menu up on and then down to "Class Manager"

This will give you a list of all the main categories of items that you have. These are the defaulted class.

If you for instance decided to sell food or drinks, you can add a different class and labeling a title of your choice such as coffee or snacks.

Step 2: Add Classes and Description

In order to add a class, you would simply click or tap on “add class” on the left of the screen. You then need to assign a unique class code. That means a code that is not currently being used somewhere else. Also you would type in the class description below that and choose one of the options. For instance, when they select that particular class, you may want to make that class taxable or you want to apply environmental fees to the class or you may not want to print a ticket for this class and the last one allowing you to popup a scan code every time you select this certain class.

Click “Save Changes” on the left pane and then click “Select Class” to see if the class that you’ve created has been added to the list.

To delete an item, select the item from the list and then press “delete item” on the left.

If want to modify a class, select the line you want to modify and then click “edit class.”

Make sure to save changes once you're done.

Step 3: Head to to Item Manager

Let’s head into the item management window by going back to the main menu, going back to the utilities menu and then down to item manager.

Select the class first and then you’ll see all the item that are within that class.

The important thing here is to have a unique item code for each of the items within the class when you add an item using the respective button on the left pane.

To edit items, click on the item from the list. Add a unique item code.

Here you have several fields where you can input description on the item. Here you have Item Description, Piece Count, Price, Class, Quantity, and Forced and Optional Upcharges.

Once you're done making changes, be sure to save before moving onto something else.

Step 4: Add Forced and Optional Upcharges

A forced upcharge means that the user is forced to make a second decision on the second description of an item.

To add a forced upcharge, click on the add button on the bottom, add a unique code that has not been used before, and then add the piece count as well a a price.

We recommend keeping the piece count and price of the original item to "0" and then adding the true value for both price and piece count under the forced upchage list. Be sure to watch the video on the tutorials page on the previous page.

For the Optional Upcharges on the next tab, again, you need to add a unique class code, add price and then description.

For the class column, you will add the class that you want to make this item to take effect. So for instance, if you want to make this optional upcharge to apply to dry cleaning and laundry, then you would put "D" and "L" respectively with not space in between.

Once you're done with the modifications, click on "Save Changes", press "OK" and this will lead you back to the main menu where you can do your next order of business.

You’ve Now Added Classes and Items and Upcharges!
